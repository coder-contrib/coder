# coder-contrib/coder

This is a fork of the [coder/coder](https://github.com/coder/coder) repository to showcase Coder's AI agent functionality.

## Using Coder with AI Agents

Coder integrates with AI agents to enhance your development workflow. Our AI integration helps you:

- Navigate and understand your codebase faster
- Automate common development tasks
- Get real-time assistance while coding

See [this issue](https://github.com/coder-contrib/coder/issues/1) for detailed documentation on how to use the integration!

## Getting Started

To start using Coder with AI agents:

1. Set up your Coder workspace
2. Configure your AI agent connection
3. Begin coding with AI assistance

For installation and setup instructions, please refer to our [documentation](https://github.com/coder-contrib/coder/issues/1).
