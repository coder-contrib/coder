# coder-contrib/coder

This is a fork of the [coder/coder](https://github.com/coder/coder) repository to showcase Coder's AI agent functionality.

## What is Coder?

Coder is a self-hosted, open-source platform for cloud development environments. It enables organizations to provision, access, and manage development workspaces in their own infrastructure.

Key features of Coder include:

- Support for any cloud, IDE, operating system, Git provider, and identity provider
- Infrastructure-as-code templates using Terraform for consistent, version-controlled workspace definitions
- Increased security through centralized source code and data management
- Improved compatibility between development, staging, and production environments
- Enhanced accessibility with browser-based IDEs and remote IDE extensions

Coder is designed to make remote development approachable for organizations of various sizes and stages of cloud-native maturity.

For more information on how to use Coder's AI agent functionality in this fork, see [this issue](https://github.com/coder-contrib/coder/issues/1).

To learn more about Coder, visit the [official Coder documentation](https://coder.com/docs/coder-oss/latest).
